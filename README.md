# 勝原PJ 受講者向けAI診断ツール デモ

マーケニスタ／コンバジョニスタ 新規受講生オンボーディングのAI診断ツール（提案用デモ・モック）。

- `index.html` … 単一HTMLで完結（外部依存なし）。
- 画面フロー：ウェルカム → 設問5ステップ → AI診断中 → 結果ダッシュボード。
- 主な機能：
  - プラン切替（フル連携版 約202万 / マジックリンク完結版 約169万）＝切替で連携機能と概算金額が変わる
  - 上位診断（「上位診断はこちら」で開く／SWOT・クロスSWOT・講師フィードバック入力欄＝生きたカルテ）
  - 診断結果（総合スコア・スキルレーダー・診断コメント・成長ロードマップ・デイリーナビ・つまずき検知）
- デザインはマーケニスタのトンマナ寄せ（オレンジ＋クリーム＋白）。※Claude design で見た目を調整中。

公開デモ：https://katsuhara-demo.vercel.app
